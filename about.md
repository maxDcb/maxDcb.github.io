# About

I am an application pentester focused on offensive security tooling, malware development, C2 engineering, and cloud security research. This blog documents personal research projects, proof-of-concepts, implementation notes, and lessons learned from building those tools.

The main thread is Exploration C2, a modular command-and-control framework built across several repositories. I also write about supporting libraries, loader research, OpenShift security graphing, and AI-assisted development workflows.

For code and project history, see [github.com/maxDcb](https://github.com/maxDcb).
